## 1. Smart Forgetting (Esquecimento Inteligente)

Diferente dos bancos de dados tradicionais onde um dado fica lá até ser deletado, o *Smart Forgetting* aplica uma **curva de decaimento**.

* **Como funciona:** Cada bloco de informação recebe um peso de relevância. Se ele não é acessado, esse peso diminui com o tempo (inspirado na Curva do Esquecimento de Ebbinghaus).
* **Vantagem:** Evita o "ruído" informacional. Isso impede que o modelo recupere fatos irrelevantes de três anos atrás que possam confundir o contexto atual.

## 2. Recency Bias (Viés de Recorrência)

O cérebro prioriza o que aconteceu recentemente porque, evolutivamente, o passado imediato tem maior probabilidade de ditar o futuro próximo.

* **Aplicação em IA:** Em vez de uma busca puramente semântica (por similaridade de significado), o sistema dá um "boost" nos resultados mais recentes.
* **Exemplo Prático:** Se você pesquisar por "reunião", o sistema prioriza a de ontem, mesmo que você tenha 500 notas sobre reuniões nos últimos 5 anos.

## 3. Context Rewriting (Reescrita de Contexto)

Esta é a parte "viva" da memória. O sistema não apenas guarda o dado bruto, ele o processa continuamente.

* **O Processo:** À medida que novas informações entram, a IA reescreve resumos de memórias antigas para incluir novas conexões.
* **Pensamento Associativo:** É aqui que a IA "conecta os pontos". Ela pode perceber que um documento sobre indicadores culturais de 2024 (seu trabalho no MinC) se conecta semanticamente a uma nova tecnologia de IA que você está estudando agora, criando um link que não existia antes.

## 4. Hierarchical Memory Layers (Camadas Hierárquicas)

Aqui entra a engenharia de infraestrutura, usando ferramentas como o **Cloudflare KV** ou **Qdrant** (que você já utiliza no seu stack).

| Camada | Tecnologia (Exemplo) | Analogia Humana | Velocidade |
| --- | --- | --- | --- |
| **Quente (Hot)** | Cloudflare KV / Redis | Memória de Trabalho | Instantânea |
| **Morna (Warm)** | Vector DB (Qdrant) | Memória de Curto Prazo | Milissegundos |
| **Fria (Cold)** | Object Storage (S3/R2) | Memória de Longo Prazo | Segundos |
