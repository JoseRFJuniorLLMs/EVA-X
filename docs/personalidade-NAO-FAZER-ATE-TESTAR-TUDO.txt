👉 **Problema**

* criatividade não óbvia.

## 3. EVA-Mind entende fatos, mas ainda não “duvida” deles

👉 **Epistemic Uncertainty explícita**

↓

  - Depth: 2 (reduzido)
  - Threshold: 0.5 (mais seletivo)

FDPN: Ativa com cuidado
  → "suicídio" (reconhecido, não explorado)

  → "Caps" (atenção psicossocial) ✓

## 4. Métricas e Monitoramento

### 4.1 Cognitive Load

**Fórmula:**

```
load = 0.3 * frequencyLoad + 0.4 * complexityLoad + 0.3 * latencyLoad

frequencyLoad = min(queries_per_minute / 10, 1.0)
complexityLoad = avg(query_complexity)
latencyLoad = min(avg_response_ms / 100, 1.0)
```

**Interpretação:**
- 0.0-0.3: Leve (normal)
- 0.3-0.6: Moderado (aumentar atenção)
- 0.6-0.8: Alto (reduzir profundidade)
- 0.8-1.0: Sobrecarga (modo shallow/silence)

### 4.2 Emotion Intensity

**Detecção simples (em produção: usar LLM):**

```
Keywords {
    "ansioso": ["preocupado", "nervoso", "ansioso", "tenso"],
    "confuso": ["confuso", "perdido", "não entendo"],
    "trauma": ["trauma", "dor", "sofrimento"],
}

intensity = sum(keyword_matches) * 0.3
```

### 4.3 Sensitivity Score

```
sensitivityKeywords = {
    "suicídio": 1.0,
    "morte": 0.9,
    "trauma": 0.9,
    "abuso": 0.95,
    "solidão": 0.7,
}

sensitivity = max(scores_of_keywords_found)

func (e *FDPNEngineV2) StreamingPrimeWithControl(userID, text string) error {
    // 1. Consulta executive
    strategy := e.executive.ModulatePriming(ctx, text, userID)
    
    // 2. Aplica estratégia
    keywords := e.extractKeywords(text)
    if len(keywords) > strategy.MaxNodes {
        keywords = keywords[:strategy.MaxNodes]
    }
    
    for _, kw := range keywords {
        e.primeKeywordWithStrategy(userID, kw, strategy)
    }
    
    // 3. Feedback
    e.executive.UpdateMetrics(text, elapsed, nodesActivated, "")
}
```
